# Global Food Price Index — краткое описание проекта и датасета

## Что за данные
- Источник: CSV `data/fao_food_price_index.csv` — официальные индексы продовольственных цен FAO (Food and Agriculture Organization).
- Содержимое: помесячные значения различных индексов (зерновые, сахар, мясо и др.). Первая колонка с датой (month/period) приводится к `DatetimeIndex`, остальные — числовые ряды.
- Тип временного ряда: ежемесячный, достаточно длинный (много лет), поэтому есть смысл использовать лаги 3–12 месяцев и годовую сезонность.

## Что умеет приложение
- Очистка и подготовка данных (`data_io.py`): автоопределение колонки даты, сортировка по времени, сохранение результатов в `results/`.
- Базовая аналитика (`analysis.py`, `menu.py`): описательная статистика, помесячная/годовая инфляция, корреляции между индексами, интерактивные графики.
- ML-блок (`ml_models.py`, `visualization.py`):
  - **Линейная авторегрессия** (лаги прошлых значений) с прогнозом вперёд.
  - **Новые бейзлайны для сравнения**: наивный прогноз, скользящее среднее, сезонный наивный, простое экспоненциальное сглаживание, плюс визуальное сравнение нескольких прогнозов на одном графике.

## Что можно подчеркнуть в презентации
- **Сезонность и тренды**: годовые колебания хорошо ловит сезонный наивный прогноз; можно показать heatmap корреляций между корзинами товаров.
- **Интерпретируемые метрики**: MAE и MAPE выводятся отдельно для train/test, есть готовые таблицы для копирования в отчёт.
- **Прогнозы на несколько месяцев вперёд**: графики из `visualization.py` объединяют историю и разные модели прогноза, что удобно для выбора стратегии.
- **Дальнейшее развитие**: добавить кросс-вариации (например, использовать корреляции между индексами как признаки), внедрить градиентный бустинг/Prophet при необходимости.

## Быстрый сценарий демо
1. Запустить `python main.py`, выбрать пункт 1, чтобы увидеть доступные индексы.
2. Пункт 4 — показать корреляции; пункт 5/6 — визуализация динамики.
3. Пункт 8 — обучить линейную AR-модель и получить прогноз.
4. Пункт 9 — сравнить простые бейзлайны и показать сводную таблицу ошибок.
